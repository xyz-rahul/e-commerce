# Next.js E-Learning Platform

## Features

-   **Server-Side Rendering (SSR):** Benefit from faster page loads and improved SEO with Next.js SSR.
-   **Responsive Design:** Enjoy a seamless user experience on various devices.
-   **Course Catalog:** Manage and showcase courses with an easy-to-use catalog system.
-   **User Authentication:** Provide secure and personalized experiences with Clerk for authentication.

## Deployment

Explore the platform at https://e-commerce-steel-eight.vercel.app

## Technologies Used

-   **Next.js:** A React framework for building server-side rendered applications.
-   **Tailwind CSS:** A utility-first CSS framework for building responsive and customizable designs.
-   **Clerk:** Integrated for user authentication.
